## 🎨 Customization Guide

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #1a3a52;      /* Dark Blue */
    --secondary-color: #d4a574;    /* Gold/Orange */
    --accent-color: #e74c3c;       /* Red */
    --text-dark: #2c3e50;          /* Dark Text */
    --text-light: #ecf0f1;         /* Light Text */
}
```
